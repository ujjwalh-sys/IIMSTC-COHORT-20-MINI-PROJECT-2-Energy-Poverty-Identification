# IIMSTC-COHORT-20-MINI-PROJECT-2-Energy-Poverty-Identification
Energy Poverty Identification

Overview

Energy poverty refers to the lack of access to reliable, affordable, and
sustainable energy services. This project analyzes datasets related to
household energy consumption and energy indicators to identify patterns
of energy poverty using data preprocessing, feature engineering, and
machine learning techniques.

Project Workflow

1.  Dataset Collection
2.  Data Preprocessing
3.  Feature Extraction and Selection
4.  Model Training
5.  Testing and Validation

Folder Description

Dataset Contains the raw datasets used in the project including
household energy consumption and energy poverty indicators.

Data Preprocessing and Features Selection Includes notebooks responsible
for: - Data cleaning - Handling missing values - Feature extraction -
Feature selection - Preparing the dataset for machine learning models

Training Testing and Validation Contains notebooks used for: - Machine
learning model training - Model testing - Validation and evaluation
